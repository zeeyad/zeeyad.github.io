Brunei Work

AI-powered Job Search Engine in Brunei

April 10, 2023

  • Ruby on Rails
  • Python
  • PostgreSQL

Brunei Work is an AI-powered Search Engine to find the latest job vacancy in Brunei. The site officially launched on 10th April 2023 by posting on the Brunei Subreddit. Follow for weekly job post highlights on the instagram

Technology Stack

  1. Front-end: HTML, CSS, Javascript, Bootstrap
  2. Back-end: Ruby on Rails, Python
  3. Database: PostgreSQL
